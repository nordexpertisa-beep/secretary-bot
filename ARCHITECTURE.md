# Секретарь — архитектура

## Назначение

Telegram-бот, который принимает сообщения и складывает их в Markdown-файлы прямо в Obsidian vault. Работает как цифровой секретарь: первое сообщение открывает новую заметку, молчание 5 минут или команда закрытия — заметка закрывается, следующее сообщение создаёт новую.

## Логика работы

### Сессия (открытие/закрытие заметки)

```
Первое сообщение
    → создаётся файл YYYY-MM-DD HH-MM-SS.md
    → запускается таймер 5 минут

Каждое следующее сообщение
    → пишется в открытый файл
    → таймер сбрасывается

Тишина 5 минут  ИЛИ  команда закрытия
    → файл закрывается
    → следующее сообщение создаст новый файл
```

### Команды закрытия

Любое из слов (регистр не важен):
`все`, `всё`, `закончил`, `закончила`, `закончить`, `закрыть`, `закрой`, `конец`, `стоп`, `хватит`, `достаточно`, `готово`, `end`, `stop`, `done`, `finish`

### Ответы на сообщения

Если пользователь ответил (reply) на сообщение в Telegram, бот находит, в каком файле лежит оригинал (через `.msg_map`), и дописывает ответ туда же. Работает даже для уже закрытых заметок.

`.msg_map` — текстовый файл вида `message_id|/path/to/file.md`, загружается при старте бота.

## Типы входящих сообщений

| Тип | Что происходит |
|-----|----------------|
| Текст | Пишется в заметку |
| Фото | Скачивается в `attachments/`, в заметке `![[photo-....jpg]]` |
| Голосовое | Скачивается `.ogg`, транскрибируется Whisper, в заметке: `![[voice-....ogg]]` + `🗣 текст` |
| Видео | Скачивается `.mp4`, в заметке `![[video-....mp4]]` |
| Документ | Скачивается с оригинальным именем, в заметке `![[filename]]` |
| Аудио | Скачивается, в заметке `![[filename.mp3]]` |
| Стикер | `[стикер: 😀]` |
| Геолокация | `📍 [lat, lon]` |

## Транскрипция голоса (Whisper)

- Библиотека: `openai-whisper`
- Модель грузится лазиво при первом голосовом сообщении (singleton, thread-safe)
- Язык зафиксирован: `ru`
- При ошибке транскрипции — в заметку пишется только аудио без текста

Смена модели без перезапуска: изменить `WHISPER_MODEL` в `.env` → `systemctl --user restart secretary-bot`

## Известные особенности

- Бот запускается на **локальной машине** (не на сервере 193.187.94.87) — сервер в России, Telegram там заблокирован на уровне провайдера
- Сервис запущен как **user-level systemd** (без sudo), стартует автоматически при логине пользователя `vitos`
- Whisper `base` при первом запуске скачивает модель ~145 MB в `~/.cache/whisper/`
